# TorneoTenis
Bienvenidos al repositorio de esta api Rest para resolver el ejercicio tecnico de entrevista de Geopagos

### **Link Productivo - Documentacion Swagger**
https://api-torneo-tenis.azurewebsites.net/swagger/index.html

### **Pruebas en Postman**
En la carpeta PruebasPostman encontraras una coleccion , donde podras realizar algunas pruebas a cada endpoint del servicio

### **Consideraciones**
Si quieres ejecutar el proyecto localmente, deberas pegar en el ConnectionStrings en el archivo de appsettings.json la cadena de conexion de una base de Datos Mongo. En entorno Productivo es tomado de azure.

### **Tecnologias usadas en el Proyecto**
- Net Core (FrameWork)
- MongoDB (Base de datos)
- XUnit  (Pruebas Unitarias)
- Postman (Pruebas de integracion)
- Swagger (Documentacion)
- Git  (Versionado)
- Azure  (Deploy)
- Entre otros..
